如何获取 Cookie

  1. 电脑浏览器打开 card.tsinghua.edu.cn，完成清华统一身份认证登录（含双因子）。
  2. 登录后按 F12 → 顶部「应用程序 / Application」标签 → 左侧 Cookies → 选 card.tsinghua.edu.cn。
  3. 找到名为 servicehall 的条目 → 双击其 Value → Ctrl+C 复制。
  4. 粘贴到上方输入框，填好学号，点「保存配置」后点同步按钮。

cookie 会过期（数小时～1 天）。过期后重复上述步骤即可，历史数据不会丢失。
